# DigiTools Platform

DigiTools is a premium, classic-style SaaS platform designed to supercharge digital workflows. It provides a curated collection of AI writing tools, design asset packs, stock media, and automation toolkits, all wrapped in a clean, professional interface.

## 🚀 Technologies Used

- **React.js 19**: Core frontend framework for a modern, component-based architecture.
- **Tailwind CSS 4**: Utility-first CSS for precise, "classic" design implementation.
- **Zustand**: Efficient state management for a seamless cart and product experience.
- **React-Toastify**: Real-time user feedback for all cart interactions.
- **Lucide React**: High-quality iconography for a professional aesthetic.

## ✨ Key Features

- **Premium Product Catalog**: A visually organized 3-column grid featuring high-quality digital assets with unique icons and detailed feature lists.
- **Integrated Cart Experience**: A persistent shopping cart system that allows users to toggle between product browsing and order management effortlessly.
- **Classic SaaS Aesthetic**: A clean, light-themed design with a primary purple color scheme, consistent spacing, and high-impact visual elements like the solid purple stats banner.
- **Interactive Feedback Loop**: Instant visual confirmation for adding products, removing items, and completing checkout through toasts and dynamic button states.
